Ownership dữ liệu

AuthServer

Users, Roles, Permissions, OpenIddict Applications, Authorizations, Tokens.

HRM

Nhân viên gốc, phòng ban, chức vụ, hợp đồng, trạng thái làm việc.

EMR

Profile y tế, chứng chỉ hành nghề, cấu hình ký, chứng thư số.

Admin Portal

Giao diện quản trị tập trung, gọi API từng hệ thống, không thay ownership.
